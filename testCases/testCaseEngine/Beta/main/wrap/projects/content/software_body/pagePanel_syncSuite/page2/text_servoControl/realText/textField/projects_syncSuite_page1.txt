Rechner, die mit Hilfe unserer Software für die Verwaltung mit der syncSuite vorbereitet werden, koennen ihr System auf einem Server sichern. Andere, vorzugsweise baugleiche, Rechner können dieses sich an diesen Rechner angleichen. 

Neben der Installation neuer Software bietet das System auch eine  Backupfunktion – bootunfaehgie Computer werden durch einen Neustart auf die letzte, funktionierende Konfiguration zurueckgesetzt. Bei Hardwareausfall muss auf der Ersatzhardware nur der syncSuite-client installiert werden , sämtliche andere Software wird dann beim Neustart installiert. Je nach Menge der Software ist die Ersatzhardware nach wenigen Minuten bis einer halben Stunde voll einsatzbereit.


Befinden sich mehrere Versionen der gleichen Software im Einsatz, so ist die syncSuite eine einfache Loesung, um die Rueckwaertskompatibilitaet zu erhalten.
Oft werden fuer unterschiedliche Projekte unterschiedliche Softwareloesungen eingesetzt. Sind Nacharbeiten an einem Projekt noetig, fuer das die Software auf Grund anderer Projekte entfernt wurde, so kann man waehrend des Bootvorgangs auf die Konfiguration seines Rechners wechseln, die die benoetigte Software bereitstellt.
